## use rollup development

```
npm i uglify-es -g
npm i rollup -g
```

```
make dev
```

```
make build
```

single js file , no dependencies

## usage

**env**

`PORT` tcp listen port

`SSLPORT` default ssl connection port

`ECHOPORT` flag run as echo server

`FORWARD_HOST` ip or domain , flag run as port forward server

`FORWARD_PORT` port
